Redmi 9 Prime

When it comes to a mobile under 10k we get pretty limited options, among them redmi 9 prime specs on paper is promising. There has been a big hike in mobile pricing recently, but redmi 9 prime looks perfect for the amount we pay.

About phone


Processor

MediaTek Helio G80

Storage

64 GB

Camera

13MP + 8MP + 5MP + 2MP (Rear Camera)
8MP (Front Camera)

Battery

5020 mAh

Display

6.53-inch

RAM

4 GB

It has 6.53 inch full HD (LCD) display.we can browse Netflix in highest quality available with this screen. We have single speaker the bottom and a 3.5mm jack. It has whooping 5020mAh battery which lasts a day or maybe an half extra as well without sweat even with heavy usage. It comes with 10W charger but mobiles supports 18w fast charger which is pretty good considering the price.It would have been better if they have went and provided an 18w charger instead because it takes almost 2-3 hours to completely charge the phone.

Even though phone is priced low they haven't compromised with design(plastic back, gorilla glass 3 protected). It runs on mediatek G80 processor which is powerful enough to run games like COD, Asphalt 9, Pubg etc.

Cameras -
Quad cam set up at the back.
13MP main camera
8MP wide lens
5MP macro lens
2MP depth sensor

Front - 8MP camera

Well for the price point it is a decent camera, so if you zoom in and check you'll notice that picture starts breaking. But colours are pretty decent.We can record using macro lens at 1080p. There's also slow motion feature.

It comes with type C cable which is pretty good. And like any other MI mobile it comes with IR blaster which can be used as remote.Ram and Rom variant being 4+64 with external SD card supported which can be extended upto 512 gb.

Well there are few resources say they lack few things like gyroscope, it doesn't work properly. Auto brightness sensors can't adjust precisely.

Do we recommend it? - well considering the price point and features it provides it's definitely worth every penny, but if you're someone who really likes to click pictures a lot then this mobile isn't for you, instead add few more bucks and go for a mid range mobile which clicks some decent pictures.

Happy Shopping!
